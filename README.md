# 🚀 FFlag Injector for Roblox

A powerful Windows tool to inject and manage **FastFlags** (FFlags) directly into a running Roblox process. Boost FPS, reduce input lag, lower ping, and customize your Roblox experience without restarting the game.

## ✨ Features

- **Real‑time injection** – Apply FFlags while Roblox is running (no restart needed).
- **Performance presets** – One‑click presets for Max FPS, Low Ping, Low‑End Devices, etc.
- **Flag detection** – Scan Roblox memory to see which FFlags are currently active.
- **JSON editor** – Paste or edit custom flag sets from external sources.
- **Auto‑applied list** – Track only the flags you have set, no clutter.
- **Auto‑update notifications** – Checks Discord for new versions (optional).

## ⚠️ Important Warning

Roblox maintains an **allow list** of supported FFlags.  
Using **FFlags that are not on the allow list** may trigger anti‑cheat systems and **carries a small risk of account suspension or ban**.  

- Presets included in this tool are considered safe (they only tweak performance and graphics settings).  
- If you add custom flags from unknown sources, you assume full responsibility.

**Use this tool at your own risk.** The author is not responsible for any enforcement actions taken against your Roblox account.

## 🎮 Quick Start

1. Download the latest `.exe` from [Releases](../../releases).
2. Run the injector **after** Roblox is open.
3. Select a preset or browse the full list of FFlags.
4. Click **Apply Flag** – the change takes effect immediately.

## 🔧 Requirements

- Windows 10 / 11
- Roblox Player (running)
